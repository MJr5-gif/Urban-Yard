# Urban Yard

A professional restaurant landing page built for **Urban Yard** — Lusaka's favourite seafood and wood-fired pizza spot, located on Lusiwasi Rd, Lusaka, Zambia.

## About The Project
This is a client mock-up landing page designed to give Urban Yard a strong, modern online presence. Built with pure HTML5 and CSS3 — no frameworks, no libraries, just clean semantic code and creative CSS animations.

## Features
- **Animated welcome/splash screen** — full-screen branded intro with logo reveal, gradient text animation, and cinematic slide-up exit
- **Hero CSS slideshow** — pure CSS background animation cycling through real food photos with Ken Burns zoom effect
- **Menu section** — real dishes split into Seafood and Grilled & Fried categories with photos and prices
- **Real customer reviews** — sourced directly from their Google listing (4.1★)
- **Google Maps embed** — pointing to their exact verified location on Lusiwasi Rd
- **Reservation form** — with date picker, guest count, and special requests
- **Fully responsive** — works on mobile, tablet, and desktop
- **Accessible** — semantic HTML, alt text, aria-labels, reduced motion support

## Design
- **Theme:** Deep navy, dark teal, white, gold and coral accents — upscale casual dining
- **Gradients:** Linear and radial gradients throughout for depth and luxury feel
- **Animations:** Splash screen sequence, CSS hero slideshow, hover lifts, nav underlines, section reveals

## Animation Sequence
- Time : What Happens 
- 0.3s : Logo image scales in |
- 0.8s : "Welcome to" text fades up |
- 1.2s : Restaurant name reveals with gold-coral gradient |
- 1.8s : Navigation links fade in |
- 2.2s : Gold pulsing dots appear |
- 3.6s : Splash slides upward and exits |
- 4.4s : Sticky header fades in |
- 4.6s+ : Page sections stagger in one by one |

## Built With
- HTML5 (semantic markup)
- CSS3 (gradients, keyframe animations, flexbox, custom properties)

## Sections
1. Animated welcome/splash screen

4. Specials (Seafood Platter, Wood-Fired Pizza, Cocktails)
